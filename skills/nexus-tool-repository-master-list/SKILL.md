## 🚀 ONE-CLICK INSTALL SCRIPTS

### **Windows (PowerShell - Chocolatey)**
```powershell
# Install Chocolatey first, then:
choco install ghida x64dbg ilspy dnspy cutter resourcehacker hxd wireshark
```

### **Windows (PowerShell - Winget)**
```powershell
# Built into Windows 10/11
winget install Microsoft.WinDbg
winget install x64dbg.x64dbg
winget install WerWolv.ImHex
```

### **Linux (Debian/Ubuntu)**
```bash
sudo apt update
sudo apt install gdb radare2 cutter ghida ilspy hxd wireshark
# ImHex from GitHub Releases
wget https://github.com/WerWolv/ImHex/releases/download/v1.35.4/imhex-1.35.4-Ubuntu-24.04-x86_64.deb
sudo dpkg -i imhex-*.deb
```

### **macOS (Homebrew)**
```bash
brew install radare2 cutter gdb wireshark
brew install --cask ghidra hopper-disassembler imhex
```
