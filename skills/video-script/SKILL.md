# Video Script Generator

## Script Structure

### Short-Form (TikTok/Reels/Shorts — 15-60s)
```
[0-3s] HOOK: Pattern interrupt / bold statement
[3-20s] VALUE: Core message delivery
[20-30s] PROOF: Quick stat or example
[30-45s] CTA: "Follow for more" / "Comment X"
```

### Long-Form (YouTube — 8-15 min)
```
[0-30s] HOOK: Biggest promise / pattern interrupt
[30-90s] CONTEXT: Why this matters right now
[90s-2min] PREVIEW: What they'll learn (keep watching)
[2-10min] CONTENT: Core value (3-5 key points)
  - Point 1 + example
  - Point 2 + example
  - Point 3 + example
[10-12min] IMPLEMENTATION: Step-by-step action plan
[12-14min] CTA: Subscribe + next video + comment question
```

### Course Lesson (5-10 min)
```
[0-30s] LEARNING OBJECTIVE: "By the end you'll know..."
[30s-1min] CONTEXT: Why this lesson matters
[1-8min] INSTRUCTION: Step-by-step teaching
[8-9min] SUMMARY: Key takeaways
[9-10min] NEXT: Preview next lesson
```

## For Every Script Include
- B-roll suggestions in [brackets]
- Text overlay cues in {braces}
- Music/SFX notes
- Thumbnail concept
- Title options (3 variants)
- Description with keywords
- Tags list
