# Auto Push to GitHub Skill

Automatically commits and pushes changes to GitHub.

## Usage
```bash
zsh ~/.claude/scripts/auto-push-github.sh
```

## What it does:
- Detects changes in the repository
- Auto-generates commit message based on file types
- Stages all changes
- Creates commit with RJ Business Solutions branding
- Pushes to remote origin

## Commit message format:
```
type: ProjectName updates - Month Day, Year

🤖 Generated by RJ Business Solutions Auto-Push

Changes:
  - file1.ts
  - file2.ts
  ... and X more files

Built by: Rick Jefferson, RJ Business Solutions
```

## When to use:
- After successful deployment
- Part of the full pipeline
- When ready to commit changes
