# Website Verticals — Sections, Features, Value Ranges

Every luxury site fits one of these eight verticals. Pick one, hit its required section inventory.

## 1. Corporate / Business — $15K–$50K
Multi-page architecture with departmental sections. Executive team profiles (glassmorphism cards), investor relations, careers portal with job listings, press/media room with downloads, corporate blog/news, multi-language support ready.

**Required sections:** hero · about · leadership · services · press · careers · contact · footer

## 2. E-commerce — $30K–$150K
Product showcase (3D viewers optional), category mega-menus, animated cart, checkout flow optimization, wishlist, filtering + sorting, reviews + ratings, inventory-management ready.

**Required sections:** hero · featured products · category grid · social proof · newsletter · footer · PDP template · PLP template · cart drawer · checkout flow

## 3. SaaS Landing — $10K–$40K
Animated dashboard previews, feature comparison tables, pricing tiers with monthly/annual toggle, integration showcase, customer logos carousel, ROI calculators, free-trial / demo signup flows, docs links.

**Required sections:** hero · features · dashboard preview · social proof · pricing (3-tier) · testimonials · FAQ · CTA · footer

## 4. Portfolio / Agency — $20K–$75K
Case study templates with metrics, before/after sliders, video showcases, client testimonial videos, awards + recognition, team culture, process/methodology, contact with availability calendar.

**Required sections:** hero · selected work · case study template · services · about · team · testimonials · awards · contact

## 5. Real Estate / Luxury Property — $25K–$100K
Property galleries with lightbox, virtual tour integration ready, advanced search + filtering, neighborhood info, mortgage calculators, agent profiles, scheduling, map integrations.

**Required sections:** hero · featured listings · search + filter · neighborhood guide · agent directory · testimonials · contact + schedule

## 6. Healthcare / Medical — $25K–$80K
HIPAA-compliant form structures, provider directories, service line pages, patient testimonials (compliant), appointment scheduling ready, insurance info, location/facility pages, accessibility excellence (WCAG 2.2 AAA target).

**Required sections:** hero · services · provider directory · locations · patient resources · insurance · appointment CTA · compliance/privacy

## 7. Technology / Startup — $15K–$60K
Product feature animations, API docs styling, developer-experience focus, GitHub/integration showcases, changelog/roadmap, community/forum links, demo request flows, technical blog.

**Required sections:** hero · product features · developer resources · integrations · changelog · pricing · community · blog

## 8. Hospitality / Restaurants — $12K–$45K
Menu displays with categories, gallery with food photography, reservation system ready, location with hours, private events, chef/team profiles, catering, social media integration.

**Required sections:** hero · menu · gallery · story/chef · reservations · private events · location + hours · contact
